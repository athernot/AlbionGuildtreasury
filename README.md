# Albion Guild Treasury Tracker

Balance tracker for Albion Online Guild.

## Quick Start

\\\ash
python -m http.server 8000
\\\

- Treasury: http://localhost:8000/index.html
- Syphon Energy: http://localhost:8000/syphon.html

## Features

- Transaction logging & management
- Import/export (Excel, CSV, PDF)
- Multi-currency support
- Offline mode (PWA)
- Dark mode
